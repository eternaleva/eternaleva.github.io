Thread.sleep()和Object.wait()

  1. sleep是Thread的静态方法，wait是Object的成员方法。
  2. sleep在休眠时间到了后，就会唤醒，wait是其他线程，在同一个对象中，调用notify()或notifyAll()。
  3. sleep使用没有任何前提，wait要求必须当前线程在持有的锁对象上调用。
  4. sleep在阻塞时不会放弃锁的持有，而wait会在阻塞时，放弃锁对象。
